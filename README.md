# ACT LAB - National Cyber Security Laboratory

ACT LAB is a premium, hands-on vulnerable web application training platform designed for **National Level** security competitions and educational excellence. It features real-world attack vectors in a safe, isolated, and highly visual environment.

## 🚀 Live Deployment
The platform is deployed and active at:
**[https://act-lab-csrf-sqli.vercel.app](https://act-lab-csrf-sqli.vercel.app)**

## 💎 Features (National Level Edition)
- **High-Fidelity UI**: Premium aesthetics with neon-cyberpunk visuals and spacious dashboard layouts.
- **In-Depth Analysis**: Every lab includes live traffic inspectors, logic visualizers, and data-flow mapping.
- **National Level CSRF**: Features an advanced Origin Verification bypass mission.
- **Automatic Identity**: Every visitor gets a unique isolated session automatically (no sign-up required).
- **SQLite Powered**: Lightweight, fast, and resets easily for training groups.

## 🛠️ Tech Stack
- **Backend**: Node.js + Express
- **Database**: SQLite (Optimized for Vercel Serverless)
- **Frontend**: Vanilla HTML/JS + Modern CSS (Inter/Orbitron Fonts)
- **Deployment**: Vercel Serverless

## 🚩 Missions Summary
1. **SQLi - Auth Bypass**: Simple login manipulation.
2. **SQLi - Union Select**: Data extraction from hidden tables.
3. **SQLi - Blind (Hard)**: Boolean-based inference with logic visualizer.
4. **XSS - Reflected**: Client-side script injection via search.
5. **XSS - Stored**: Persistent guestbook payload injection.
6. **CSRF (Elite)**: Bypass National-Standard Origin Verification.
7. **IDOR**: Insecure direct object manipulation in profile IDs.
8. **Open Redirect**: Navigation-based phishing simulation.
9. **Privilege Escalation**: Role-based authorization bypass via cookies.
